## supabase branches disable

Deshabilitar la ramificación de vista previa para el proyecto vinculado.

### Uso

```
supabase branches disable
```

### Banderas

- **--experimental**  
    _Requerido_  
    habilitar características experimentales
    
- **--project-ref <string>**  
    _Opcional_  
    Referencia del proyecto de Supabase.
